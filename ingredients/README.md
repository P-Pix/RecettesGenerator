# Ingredients

Le dossier ingredients contient un fichier pour chaque ingredients.

## Configuration

Le fichier de configuration est un fichier JSON qui contient les informations suivantes :

```json
{
    "kcal": 100,
    "transforme" : false,
    "matiere grasse": 3.2,
    "glucide": 44,
    "proteine": 4.8,
    "lipide": 0.8,
    "sodium": 0.7,
    "fibres": 0.3,
    "sel": 0.1,
    "vitamine" : {
        "a": 0.1,
        "b1": 0.1,
        "b2": 0.1,
        "b3": 0.1,
        "b5": 0.1,

}
```
